> Privacy-first safety planning that runs ENTIRELY ON-DEVICE. No cloud. No data trail. Because any digital footprint could get someone killed.

## The Solution

**SafetyPlanner** — Personalized escape plans with shelter availability, legal aid, financial resources (ON-DEVICE ONLY)\n**StalkerwareDetector** — Detect and safely remove tracking software from phones\n**EvidenceVault** — Encrypted documentation of abuse with chain-of-custody metadata\n**DangerAssessor** — Validated lethality risk scoring (Campbell Danger Assessment)\n**ResourceConnector** — Shelters, legal aid, financial assistance, counseling by location

## Agents

MANDATORY: National DV Hotline 1-800-799-7233. CRITICAL: This tool MUST run on-device with zero cloud dependency. Any data leak could be lethal.
